# Learn Portuguese

> **Offline Android app for Danish users learning European Portuguese**

---

# Overview

Learn Portuguese is an Android application developed in **Android Studio (Kotlin)**.

The application is designed for Danish-speaking beginners who want to learn **European Portuguese (Português Europeu)** through practical conversations, repetition and simple quizzes.

Version 1 is a completely **offline application**.

No Internet connection is required.

No login is required.

No user account is required.

---

# Objectives

The primary objective is to make learning Portuguese enjoyable and structured.

The application should allow the user to:

* complete lessons in order
* repeat previous lessons
* listen to Portuguese pronunciation
* complete quizzes
* track learning progress
* continue where they left off

---

# Target audience

The application is intended for:

* Danish tourists
* Business travellers
* Students
* Expats
* Beginners with no previous knowledge of Portuguese

The application focuses exclusively on **European Portuguese**.

---

# Version 1 Features

* 10 lessons
* 10 dialogues in each lesson
* More than 200 useful Portuguese expressions
* Danish translations
* Grammar notes
* Vocabulary list
* Pronunciation using Android Text-to-Speech
* Lesson quizzes
* Local progress tracking
* Offline operation
